    format-detection 속성을 사용하여 연속된 숫자의 자동 링크(전화걸기) 기능을 제한함

    세이프 영역 호환

      function setVh() {
  let vh = window.innerHeight * 0.01;
  document.documentElement.style.setProperty("--vh", `${vh}px`);
  }

   max-height: calc(var(--vh, 1vh) * 100 - 70px);
