i30 Fastback N

정제된 아드레날린

Overview

기품 있는 반항아

차량의 다운포스, 항력, 공기역학 간 최적의 비율은 '형태는 기능을 따른다'라는 원칙을 바탕으로 개발되어 최고의 주행 경험을 제공합니다. 새로운 5도어 쿠페는 더 길고 슬림해진 뒤태로 눈에 띄는 실루엣과 스포티한 외관을 자랑합니다.

i30 N Fastback의 스포티함
Performance

뛰어난 조력자

표준 모델 대비 프런트 액슬의 측면 강성이 강화되고 중량이 감소되었습니다. N 파워 센스 액슬은 프런트 액슬에서 운전자에게 빠르고 정확한 피드백을 전달하고, 접지력을 향상해 엔진 출력을 도로에 효과적으로 전달합니다.

Design

우아한 야성미

우아하면서도 세련된 디자인에는 N 특유의 입증된 드라이빙 재미 요소를 갖췄습니다. 이는 미소를 자아내는 N만의 특별한 감성입니다.

전면

V자형 주간 주행등이 탑재된 새로운 LED 헤드램프

측면

측면에서 확인되는 새로운 하이라이트 중 하나는 현대 최초의 19인치 단조 알로이 휠입니다.

후면

더 크게, 더 탁월하게

i30 Fastback N의 외형 디자인

모든 각도에서 우아한

새로워진 i30 Fastback N의 인테리어는 모든 접점에서 차량과 드라이버를 연결합니다. 이러한 연결성은 자동차를 진정으로 사랑하는 사람들을 전적으로 고려하여 설계되었습니다.

N 스티어링 휠

독점적인 블루 스티치와 N 로고가 특징인 가죽 N 스티어링 휠은 N 모드 및 N 그린 시프트(NGS, N Grin Shift) 전용 버튼과 같은 주행 성능 기능을 간편하게 이용할 수 있도록 제공합니다.

N 라이트 시트 - 전문 레이서처럼 탑승.

프리미엄 가죽과 Alcantara® 소재로 마감된 가벼운 고성능 모노폼 시트는 반짝이는 N 로고가 각인되어 있으며, 눈에 띄는 볼스터로 탁월한 측면 지지력을 제공합니다.
