‘녹색 지옥’에서 승리하는 방법

뉘르부르크링 24시 레이스는 154개의 코너가 있는 25km의 험난한 서킷에서 차량의 한계를 시험합니다.
현대 N은 이 악명 높은 레이스에 성공하겠다는 각오로 도전하여, 9년 연속 결승선에 도착했습니다.

뉘르부르크링에서 연마

  1. 2016

    N 파워트레인이 장착된 i30

    N 브랜드 출시 및 2.0리터 터보 엔진 테스트 사전 공지

  2. 2017

    i30 N

    N 최초 모델인 i30 N의 최종 테스트

  3. 2018

    i30 N TCR

    i30 N TCR의 성능과 내구성 테스트

  4. 2019

    Veloster N TCR
    i30 N TCR
    i30 Fastback N

    다수의 N 차량 라인업으로 첫 참가, 성공적인 성능 입증

  5. 2020

    Veloster N TCR
    i30 N TCR
    i30 Fastback N

    N 최초 UN SP3T 클래스 우승으로 내구성 입증

  6. 2021

    아반떼 N TCR
    i30 N TCR
    i20 N

    N 최초 UN SP3T 클래스 우승으로 내구성 입증

  7. 2022

    i30 N TCR
    아반떼 N TCR
    i30 N 컵 레이싱카

    두 아반떼 N TCR 모두 클래스에서 1, 2위 차지. i30 N 컵 레이싱카가 해당 카테고리에서 가장 빠른 자동차임을 입증

  8. 2023

    아반떼 N TCR
    i30 패스트백 N 컵카

    두 아반떼 N TCR 모두 클래스에서 1, 2위 차지. 악명 높은 레이스에서 3회 연속 1, 2위 달성

  9. 2024

    아반떼 N TCR
    i30 패스트백 N컵 자동차

    아반떼 N이 TCR 클래스에서 1, 2, 3위를 차지하여 4년 연속 우승 달성

전설적인 레이싱 트랙, 전설적인 레이스

N 브랜드와 뉘르부르크링의 관계는 깊습니다. N 브랜드의 이름은 현대자동차의 기술 센터가 운영되는 뉘르부르크링(Nürburgring)과 남양(Namyang) 연구소에서 따왔습니다.
‘녹색 지옥’으로 알려진 뉘르부르크링은 N 브랜드 모든 고성능 차량에 도전적인 시험장입니다.
