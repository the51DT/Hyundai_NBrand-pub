N 커뮤니티 생성 안내

N 커뮤니티를 생성하고 회원들과 다양한 미션 참여와 특별한 경품을 받아보실 수 있습니다.

N 커뮤니티 생성

1) 고객별로 1개의 N 커뮤니티만 생성가능하며, 생성 시 해당 커뮤니티장 권한을 가집니다.

2) N 커뮤니티 생성한 경우 다른 커뮤니티 가입 불가합니다.

3) 다른 N 커뮤니티에 가입된 경우, 커뮤니티 생성 불가합니다.

4) 커뮤니티 생성 시 신청즉시 또는 커뮤니티장의 승인 하에 가입 진행됩니다.

N 커뮤니티 수정

1) 커뮤니티 아이콘, 커뮤니티 링크, 가입승인방식 수정 가능합니다.

N 커뮤니티 미션/포인트/경품

1) N 커뮤니티를 생성하면, N 커뮤니티장과 N 커뮤니티 미션 모두 참여 가능합니다.

2) N 커뮤니티 미션 참여완료로 획득한 커뮤니티 포인트로 적립됩니다.

3) 지급받은 포인트는 경품 교환 가능합니다.

4) 경품 교환은 N 커뮤니티장만 신청가능합니다.

5) 지급받은 커뮤니티 경품은 커뮤니티 내부에서 분배하며, N브랜드 서비스는 관여하지 않습니다.

N 커뮤니티 가입 승인

1) 커뮤니티 가입방식이 승인방식일 경우, 커뮤니티장의 승인 하에 커뮤니티 가입 승인됩니다.
